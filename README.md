# Data_drifters
Test Repo for Agentic AI Merathon

## Demo Python Files

This repository contains a few basic Python files for demonstration and testing purposes:

### 1. hello_world.py
A simple Hello World program that prints greeting messages.

**Usage:**
```bash
python hello_world.py
```

### 2. calculator.py
A basic calculator with arithmetic operations (add, subtract, multiply, divide).

**Usage:**
```bash
python calculator.py
```

### 3. data_utils.py
Basic data utilities for list operations including sum, average, max, and min calculations.

**Usage:**
```bash
python data_utils.py
```

## Running the Demo Files

To run any of the Python files, simply execute:
```bash
python <filename>.py
```

For example:
```bash
python hello_world.py
python calculator.py
python data_utils.py
```
